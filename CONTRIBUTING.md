# Contributing to Ticker

Thanks for stopping by! A quick note on how this project runs:

Ticker is a **personal, solo-maintained** open-source project by
[Steve Ye / 예상국](https://stevelabs.io). It's public so you can run it, read
it, and learn from it — I'm just not really set up to take on contributors, so
**I generally don't merge external pull requests**. Nothing personal; it just
keeps the project something one person can maintain and keep small.

What's genuinely helpful:

- **Bug reports** — a clear, reproducible issue (version, config, steps,
  expected vs actual) is the most useful thing you can send.
- **Forks** — Ticker is [Apache-2.0](LICENSE). Want it to do more, or go a
  different direction? Fork it freely, no permission needed. That's what the
  license is for.

For anything security-sensitive, please reach out privately via the contact
links on <https://stevelabs.io> rather than opening a public issue.

— Steve
